 HOST Returns the server host name of the connection.

 Usage: hostName=pgmex('host',dbConn)

 INPUT:
   regular:
     dbConn: uint32 or uint64 [1,1] - C pointer to a PGconn structure
 OUTPUT:
   regular:
     hostName: char [1,N] - host name
