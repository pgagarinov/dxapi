DB Returns the database name of the connection.

 Usage: dbName=pgmex('db',dbConn)

 INPUT:
   regular:
     dbConn: uint32 or uint64 [1,1] - C pointer to a PGconn structure
 OUTPUT:
   regular:
     dbName: char [1,N] - database name
